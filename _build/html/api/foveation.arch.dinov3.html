foveation.arch.dinov3.resample_patch_embed_conv(conv: Conv2d, target_hw=(8, 8), mode: str = 'bicubic', align_corners: bool = True, preserve_kernel_norm: bool = False) → Conv2d[source]

Resample a patch-embedding Conv2d’s kernels from (kH, kW) -> target_hw and return a NEW Conv2d with kernel_size=stride=target_hw. Supports both upsampling and downsampling. Assumes:

  • stride == kernel_size (patch embedding)

  • padding == 0, groups == 1, dilation == 1
